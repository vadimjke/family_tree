# family_tree_parser

Type `npm i` to install dependencies
Type `npm run dev` in cmd to start script

There will be `parsed.json` file in folder

### JSON structure:
+ personsListObj
    + 0

						+ context
						+ className (rodstv)

						+ context
						+ className (person2)
					+ 2
					+ ...

				+ 2
				+ ...

				+ 0

				+ 2
